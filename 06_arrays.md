1. Concept

An array in C# is used to store multiple values of the same type in a single variable. Instead of creating many variables, you can use one array to store a list of values.

Each value in the array is accessed using an index starting from 0. Arrays are useful when you need to work with collections of data like numbers or names.

2. Key C# Syntax

```csharp
int[] numbers = { 10, 20, 30, 40 };

Console.WriteLine(numbers[0]);
Console.WriteLine(numbers[2]);
